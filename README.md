# DESAFIO-GITHUB-MARKDOWN
Vamos praticar?! Te desafio a criar uma conta no GitHub, adicionar um repositório público com o seu README.md utilizando o GitHub.dev. Adicione colaboradores, e pratique formatação Markdown para instruções e descrição do projeto. Realize alterações, Commits, abra issues e pull requests para experimentar os processos de colaboração no Git.
